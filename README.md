Notebook ini dibuat untuk mengambil (scraping) data tweet dari platform Twitter/X menggunakan Twitter API v2 melalui library Tweepy. Fokus scraping adalah pada tweet yang mengandung hashtag #DigitalisasiPembelajaran dan yang berkaitan dengan Indonesia Cerdas.

Berikut langkah langkah nya :
1. Instalasi & Import Library
2. Menyiapkan Twitter API
3. Membuat Client Twitter API v2
4. Melakukan Pencarian Tweet (Scraping) disini menggunakan #DigitalisasiPembelajaran sebagai bahan scraping nya
5. Menyimpan Hasil Scraping ke Dalam DataFrame
6. Lalu data disimpan kedalam file bernama "tweets_DigitalisasiPembelajaran.txt"
